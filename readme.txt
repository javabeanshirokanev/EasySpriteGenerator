This is the easily program for generating sprite animations.
Your must to load a few pictures and to input some parameters.

Tutorial:
1. Run the program (start .exe file or run the project from Visual Studio).
2. Choose some pictures in .png format.
3. Input the delay - this value characterize the frame count between the nearest frames. 1 - all frames will
be include to animation. 2 - each second frame will be missed. 3 - each third frame will be missed. etc.
4. Input the size of result frames. The program will be resize each frame depends on inputed values.
5. Input the row count. The program will be formed the big picture consists of generated frames. You can
choose the row count of result table. Each cell of it will be have separate frame.
6. Wait when the program finished calculations.
